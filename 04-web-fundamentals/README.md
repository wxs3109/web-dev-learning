# Web Fundamentals

Cross-cutting web development concepts that apply across all pillars. These modules cover how browsers work, web performance, build tooling, testing theory, and development workflows. This section can be studied at any time as reference material.

## Modules

| # | Module | Topics |
|---|---|---|
| 01 | [Browser & DevTools](01-browser-and-devtools/) | How browsers work, DevTools tabs, storage APIs |
| 02 | [Web Performance](02-web-performance/) | Core Web Vitals, critical rendering path, caching |
| 03 | [Build Tools & Bundling](03-build-tools-and-bundling/) | npm, bundlers, transpilation, linting |
| 04 | [Testing Fundamentals](04-testing-fundamentals/) | Testing pyramid, TDD, mocking, coverage |
| 05 | [Version Control & Workflow](05-version-control-workflow/) | Git branching, PRs, conventional commits, CI/CD |

## Note
These topics use `notes/` folders instead of `examples/` and `exercises/` since they are more conceptual. Written notes and references are the primary output.
