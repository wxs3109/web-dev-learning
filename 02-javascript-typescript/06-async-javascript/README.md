# Async JavaScript

## Learning Objectives
- Understand the callback pattern and its limitations
- Create and chain promises using then, catch, and finally
- Use Promise combinators: Promise.all, Promise.race, Promise.allSettled, and Promise.any
- Write asynchronous code with async/await syntax
- Handle errors in async code with try/catch
- Explain the event loop, microtasks, and macrotasks
- Use timing functions: setTimeout, setInterval, and requestAnimationFrame

## Key Concepts
JavaScript is single-threaded but non-blocking, thanks to the event loop. Asynchronous programming allows long-running operations (network requests, timers, file I/O) to execute without freezing the UI. Promises and async/await provide structured patterns for managing asynchronous workflows.

## Topics Covered
- Callbacks
- Promises: then, catch, finally
- Promise combinators: Promise.all, Promise.race, Promise.allSettled, Promise.any
- async/await
- Error handling with try/catch in async code
- Event loop
- Microtasks vs macrotasks
- setTimeout, setInterval, requestAnimationFrame
