# JavaScript & TypeScript

JavaScript fundamentals through advanced TypeScript. This pillar covers core JavaScript (variables, functions, DOM, async, OOP), then builds into TypeScript's type system, generics, and advanced patterns.

## Modules

| # | Module | Topics |
|---|---|---|
| 01 | JS Fundamentals | Variables, types, operators, control flow |
| 02 | Functions & Scope | Declarations, arrows, closures, hoisting, `this` |
| 03 | Objects & Arrays | Destructuring, array methods, spread/rest, JSON |
| 04 | DOM Manipulation | querySelector, creating/modifying elements, classList |
| 05 | Events | Listeners, bubbling/capturing, delegation, debounce |
| 06 | Async JavaScript | Promises, async/await, event loop, microtasks |
| 07 | Fetch & HTTP | Fetch API, CORS, REST concepts, AbortController |
| 08 | ES6+ Features | Modules, Map/Set, Symbol, iterators, generators |
| 09 | OOP & Classes | Prototypes, ES6 classes, private fields, design patterns |
| 10 | Error Handling & Debugging | Error types, try/catch, DevTools, debugging strategies |
| 11 | TypeScript Basics | tsconfig, basic types, annotations, type narrowing |
| 12 | TypeScript Interfaces & Types | Interfaces vs types, utility types, mapped/conditional |
| 13 | TypeScript Generics | Generic functions/classes, constraints, keyof |
| 14 | TypeScript Advanced | Decorators, .d.ts files, discriminated unions, strict mode |

## Prerequisites
Complete 01-html-css modules 01-08 (HTML fundamentals through responsive design) before starting.

## Recommended Order
Complete JS modules 01-10 sequentially. TypeScript modules 11-14 build on JS knowledge and are prerequisite for Angular.
