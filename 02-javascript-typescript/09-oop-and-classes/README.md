# OOP & Classes

## Learning Objectives
- Understand JavaScript's prototypal inheritance model
- Define classes using ES6 class syntax with constructors
- Use static methods and properties
- Implement getters and setters
- Encapsulate data with private fields (#)
- Extend classes using extends and super
- Apply mixins for composition-based reuse
- Recognize and implement common design patterns: singleton, observer, and factory

## Key Concepts
JavaScript uses prototypal inheritance rather than classical inheritance, but ES6 class syntax provides a familiar, cleaner abstraction over prototypes. Understanding both the underlying prototype chain and the class syntax is essential for building maintainable, extensible object-oriented code.

## Topics Covered
- Prototypal inheritance
- ES6 class syntax
- Constructor
- Static methods and properties
- Getters and setters
- Private fields (#)
- Class inheritance: extends and super
- Mixins
- Design patterns: singleton, observer, factory
