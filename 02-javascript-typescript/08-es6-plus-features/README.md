# ES6+ Features

## Learning Objectives
- Use JavaScript modules with import/export (default and named exports)
- Work with Symbol for creating unique identifiers
- Understand iterators and the for...of protocol
- Write generator functions using function* and yield
- Use Map, Set, WeakMap, and WeakSet for specialized data storage
- Apply Proxy and Reflect for metaprogramming
- Create tagged template literals
- Load modules on demand with dynamic import()
- Deep clone objects with structuredClone

## Key Concepts
ES6 (ECMAScript 2015) and subsequent revisions introduced major features that modernized JavaScript. Modules enable code organization, new data structures (Map, Set) solve limitations of plain objects and arrays, and metaprogramming features (Proxy, Reflect, Symbol) unlock advanced patterns.

## Topics Covered
- Modules: import/export, default and named exports
- Symbol
- Iterators and for...of
- Generators (function*)
- Map, Set, WeakMap, WeakSet
- Proxy and Reflect
- Tagged template literals
- Dynamic import()
- structuredClone
