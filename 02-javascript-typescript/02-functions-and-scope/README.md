# Functions & Scope

## Learning Objectives
- Distinguish between function declarations, expressions, and arrow functions
- Use default, rest, and spread parameters effectively
- Understand closures and lexical scope
- Explain hoisting behavior for variables and functions
- Work with higher-order functions and the `this` keyword

## Key Concepts
Functions are first-class citizens in JavaScript, meaning they can be assigned to variables, passed as arguments, and returned from other functions. Understanding scope, closures, and how `this` behaves in different contexts is critical for writing predictable JavaScript code.

## Topics Covered
- Function declarations vs expressions
- Arrow functions
- Parameters: default, rest, spread
- Closures
- Hoisting
- Lexical scope
- IIFE (Immediately Invoked Function Expressions)
- Higher-order functions
- The `this` keyword
