# Objects & Arrays

## Learning Objectives
- Create and manipulate object literals and use computed properties
- Apply destructuring to objects and arrays
- Use essential array methods including map, filter, reduce, find, some, every, flat, and flatMap
- Understand the spread/rest operator for both objects and arrays
- Differentiate between shallow and deep copies
- Parse and stringify JSON data

## Key Concepts
Objects and arrays are the primary data structures in JavaScript. Mastery of destructuring, modern array methods, and the spread/rest syntax enables writing concise, readable code. Understanding reference types and copy behavior prevents common bugs.

## Topics Covered
- Object literals
- Destructuring (objects and arrays)
- Computed properties
- Array methods: map, filter, reduce, find, some, every, flat, flatMap
- Spread/rest with objects and arrays
- Shallow vs deep copy
- JSON parse/stringify
