# Events

## Learning Objectives
- Attach and remove event listeners using addEventListener and removeEventListener
- Work with the event object and its properties
- Understand event bubbling and capturing phases
- Implement event delegation for efficient event handling
- Use preventDefault and stopPropagation appropriately
- Create and dispatch custom events with CustomEvent
- Handle keyboard, mouse, and touch events
- Apply debounce and throttle techniques to optimize event handlers

## Key Concepts
Events are the primary mechanism for user interaction in web applications. The browser fires events in response to user actions (clicks, key presses, touches) and system events (page load, resize). Understanding event propagation, delegation, and performance optimization patterns is essential for building interactive interfaces.

## Topics Covered
- addEventListener / removeEventListener
- Event object
- Bubbling and capturing
- Event delegation
- preventDefault / stopPropagation
- Custom events (CustomEvent)
- Keyboard, mouse, and touch events
- Debounce and throttle
