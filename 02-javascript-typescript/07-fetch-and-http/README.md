# Fetch & HTTP

## Learning Objectives
- Use the Fetch API to make GET, POST, PUT, and DELETE requests
- Work with Request and Response objects and headers
- Implement robust error handling patterns for network requests
- Cancel in-flight requests with AbortController
- Understand CORS basics and how browsers enforce same-origin policy
- Apply REST API concepts when interacting with JSON APIs

## Key Concepts
The Fetch API provides a modern, promise-based interface for making HTTP requests from the browser. Understanding request/response structure, HTTP methods, headers, and error handling is essential for building applications that communicate with backend services and third-party APIs.

## Topics Covered
- Fetch API: GET, POST, PUT, DELETE
- Request and Response objects
- Headers
- Error handling patterns
- AbortController
- CORS basics
- REST API concepts
- Working with JSON APIs
