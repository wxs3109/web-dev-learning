# JS Fundamentals

## Learning Objectives
- Understand the differences between var, let, and const
- Identify and work with all JavaScript data types
- Use operators and understand type coercion
- Write control flow logic with if/else, switch, and loop constructs
- Apply modern syntax features like template literals, nullish coalescing, and optional chaining

## Key Concepts
JavaScript is a dynamically typed, interpreted language that forms the foundation of web development. Mastering variables, types, operators, and control flow is essential before moving to functions, objects, and more advanced patterns.

## Topics Covered
- Variables: var, let, const
- Data types: string, number, boolean, null, undefined, symbol, bigint
- Operators (arithmetic, comparison, logical, assignment)
- Type coercion (implicit and explicit)
- Control flow: if/else, switch, for, while, for...of
- Template literals
- Nullish coalescing (??)
- Optional chaining (?.)
