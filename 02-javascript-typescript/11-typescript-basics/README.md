# TypeScript Basics

## Learning Objectives
- Install TypeScript and configure a project with tsconfig.json
- Use basic types: string, number, boolean, array, tuple, enum, any, unknown, void, and never
- Apply type annotations and understand type inference
- Use type assertions with the `as` keyword
- Combine types with union and intersection operators
- Work with literal types for precise value constraints
- Narrow types using type guards: typeof, instanceof, and the in operator

## Key Concepts
TypeScript is a typed superset of JavaScript that compiles to plain JavaScript. It adds static type checking, which catches errors at compile time rather than runtime. TypeScript's type system is structural (duck-typed) rather than nominal, and it provides powerful inference that minimizes the need for explicit annotations.
