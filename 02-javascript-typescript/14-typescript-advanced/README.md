# TypeScript Advanced

## Learning Objectives
- Use decorators for class, method, property, and parameter decoration
- Create and consume declaration files (.d.ts) for untyped libraries
- Understand the difference between namespaces and modules
- Configure and leverage strict mode options for maximum type safety
- Model state with discriminated unions
- Implement exhaustiveness checking using the never type
- Build type-safe event emitters
- Apply declaration merging for extending existing types

## Key Concepts
Advanced TypeScript features enable sophisticated type-level programming and integration with the broader JavaScript ecosystem. Decorators add metadata-driven behavior, declaration files bridge untyped libraries, and patterns like discriminated unions and exhaustiveness checking ensure that your code handles every possible case at compile time.

## Topics Covered
- Decorators: class, method, property, parameter
- Declaration files (.d.ts)
- Namespaces vs modules
- Strict mode options
- Discriminated unions
- Exhaustiveness checking (never type)
- Type-safe event emitters
- Declaration merging
