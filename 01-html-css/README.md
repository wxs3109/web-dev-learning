# HTML & CSS

The foundation of all web user interfaces. This pillar covers HTML document structure, semantic markup, forms, accessibility, and CSS from basics through advanced layout, animation, and architecture.

## Modules

| # | Module | Topics |
|---|---|---|
| 01 | [HTML Fundamentals](01-html-fundamentals/) | Document structure, DOCTYPE, head/meta, basic tags |
| 02 | [Semantic HTML](02-semantic-html/) | header/nav/main/section/article/footer, landmark roles |
| 03 | [Forms & Validation](03-forms-and-validation/) | Input types, labels, fieldsets, native validation |
| 04 | [Media & Embedding](04-media-and-embedding/) | Images, srcset/picture, video, audio, iframe, SVG |
| 05 | [Accessibility](05-accessibility/) | ARIA, alt text, keyboard nav, focus management, WCAG |
| 06 | [CSS Fundamentals](06-css-fundamentals/) | Selectors, specificity, cascade, box model, units |
| 07 | [CSS Layout](07-css-layout/) | Positioning, Flexbox, Grid, multi-column |
| 08 | [Responsive Design](08-responsive-design/) | Media queries, mobile-first, container queries, clamp() |
| 09 | [CSS Animations & Transitions](09-css-animations-transitions/) | Transitions, keyframes, transforms, performance |
| 10 | [CSS Variables & Theming](10-css-variables-and-theming/) | Custom properties, dark/light themes, design tokens |
| 11 | [Sass Preprocessor](11-sass-preprocessor/) | Variables, nesting, mixins, partials, functions |
| 12 | [CSS Architecture](12-css-architecture/) | BEM, OOCSS, utility-first concepts, stylesheets organization |

## Recommended Order

Complete modules 01-08 before starting JavaScript. Modules 09-12 can be done in parallel with JavaScript/TypeScript study.
