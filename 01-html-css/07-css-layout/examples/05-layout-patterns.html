Common Layout Patterns（常见布局模式）

1. Multi-column 多列文本布局

column-count: 3; column-gap: 24px

适合长段文字排版，类似报纸杂志的分栏效果。

CSS Grid 和 Flexbox 并不是用来做文本分栏的。如果你需要把一大段文字分成多列显示， 应该用 CSS Multi-column Layout。只需要设置 column-count 就能把内容自动分成指定 列数。column-gap 控制列间距，column-rule 可以加分隔线。浏览器会自动处理文字 的换行和列平衡。这种布局方式非常适合文章、新闻、博客等长文本内容。当窗口变窄时， 可以配合 media query 减少列数。Multi-column layout is perfect for text-heavy content like articles and blog posts. The browser handles text reflow automatically across columns.

2. Holy Grail 圣杯布局（Grid 实现）

Header + Nav + Content + Aside + Footer

网页最经典的布局模式，用 grid-template-areas 代码最直观。

Header（头部，横跨所有列）
Nav 导航栏
（固定 180px）
Main Content 主内容区域
（flex: 1fr，占满剩余空间）
Aside 侧边栏
（固定 150px）
Footer（页脚，横跨所有列）

3. Sidebar 侧边栏布局

Flexbox 版本

侧边栏 width: 220px + flex-shrink: 0，主内容 flex: 1。

4. Card Grid 响应式卡片网格

repeat(auto-fit, minmax(200px, 1fr))

拖动浏览器窗口改变宽度，看看卡片如何自动重排。无需 media query！
