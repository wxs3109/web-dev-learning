Flexbox Item Properties（Flex 项目属性）

flex-grow（扩展比例）

全部 flex-grow: 0（默认，不扩展）

grow:0
grow:0
grow:0

全部 flex-grow: 1（等分多余空间）

grow:1
grow:1
grow:1

flex-grow: 1, 2, 1（中间拿双倍）

grow:1
grow:2（我更宽）
grow:1

flex-shrink（收缩比例）

flex-shrink: 0, 1, 3（第一个不收缩，第三个收缩最快）

shrink:0（不缩）
shrink:1
shrink:3（缩最多）

flex-basis（初始大小）

flex-basis: auto vs 200px vs 0

basis:auto + grow:1
basis:200px + grow:1
basis:0 + grow:1

flex 简写（推荐写法）

flex: 1 — 三个项目等分空间

flex:1
flex:1
flex:1

flex: 1 + flex: 2 — 中间占双倍

flex:1
flex:2
flex:1

flex: none vs flex: auto

flex:none（不伸缩）
flex:auto（按内容扩展，我的文字比较多所以我更宽）
flex:1

align-self（单个项目的交叉轴对齐）

容器 align-items: flex-start，但个别项目覆盖

默认 (flex-start)
align-self: center
align-self: flex-end
align-self: stretch

order（视觉排列顺序）

DOM 顺序: A→B→C→D，但 order 改变了视觉位置

A (order:3)
B (order:1)
C (order:0 默认)
D (order:-1)

视觉顺序变成：D(-1) → C(0) → B(1) → A(3)
注意：order 只改变视觉顺序，不影响 Tab 键导航和屏幕阅读器。
