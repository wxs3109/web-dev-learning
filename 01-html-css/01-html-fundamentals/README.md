# HTML Fundamentals

## Learning Objectives
- Understand the purpose of DOCTYPE and how browsers render HTML
- Build a well-structured HTML document with proper head and body sections
- Use meta tags to control page behavior and provide metadata
- Create content with headings, paragraphs, lists, and links
- Apply common HTML tags to mark up text content correctly

## Key Concepts
HTML (HyperText Markup Language) is the foundation of every web page. It defines the structure and meaning of content on the web. Understanding how to write valid, well-structured HTML is the first step in web development. This module covers the core building blocks that every HTML document requires.

## Topics Covered
- The DOCTYPE declaration and its role in standards mode
- The `<html>`, `<head>`, and `<body>` element hierarchy
- Essential meta tags (`charset`, `viewport`, `description`)
- Heading levels (`<h1>` through `<h6>`) and their hierarchy
- Paragraphs (`<p>`) and line breaks (`<br>`)
- Ordered lists (`<ol>`), unordered lists (`<ul>`), and list items (`<li>`)
- Anchor links (`<a>`) and the `href` attribute
- Inline elements: `<strong>`, `<em>`, `<span>`, `<code>`
- Block elements: `<div>`, `<blockquote>`, `<pre>`
- HTML comments
