Subsection (h3)

Headings should follow a logical hierarchy. Don't skip levels (e.g., don't jump from h2 to h4). This matters for accessibility — screen readers use heading levels to navigate the page.

Inline Text Formatting

Use strong for important text (renders bold by default). Use em for emphasis (renders italic by default). These carry semantic meaning, not just visual styling.

Use <code> for inline code like console.log(). Use span as a generic inline container — it has no meaning on its own, but is useful for styling with CSS later.
