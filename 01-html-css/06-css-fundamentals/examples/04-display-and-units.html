Display and CSS Units

display: block

Block item A（我本来是 inline，现在变成 block 了） Block item B（我也独占一行）

display: inline

Inline A（我本来是 block）
Inline B（现在和 A 挤在一行了）

display: inline-block

Inline-block A Inline-block B

display: none

You should not see this text.（你不应该看到这段话）

Units（单位）

1.25em 文字（相对于父元素字体大小：16 × 1.25 = 20px）

1.25rem 文字（相对于根元素字体大小：16 × 1.25 = 20px）

240px 宽（绝对单位，固定不变）
50% 宽（父元素宽度的一半，拖动窗口试试）
40vw 宽（视口宽度的 40%）
20vh 高（视口高度的 20%）
30ch 宽（约 30 个字符宽）
