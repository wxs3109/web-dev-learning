# Pipes

## Learning Objectives
- Use built-in pipes to transform data in templates
- Create custom pipes for application-specific transformations
- Explain the difference between pure and impure pipes and their performance implications
- Chain multiple pipes together
- Pass parameters to pipes

## Key Concepts
Pipes transform displayed values within templates without modifying the underlying data. Angular provides a rich set of built-in pipes for common transformations like date formatting, currency display, and case conversion. Custom pipes encapsulate reusable transformation logic. Pure pipes only re-evaluate when their input reference changes, while impure pipes run on every change detection cycle, making the pure/impure distinction important for performance.

## Topics Covered
- Built-in pipes (date, currency, percent, async, json, slice, titlecase, uppercase, lowercase)
- Creating custom pipes
- Pure vs impure pipes
- Pipe chaining
- Parameterized pipes
