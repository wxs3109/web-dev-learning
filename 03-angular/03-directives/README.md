# Directives

## Learning Objectives
- Use structural directives (*ngIf, *ngFor, *ngSwitch) to control DOM rendering
- Apply the new control flow syntax (@if, @for, @switch) introduced in recent Angular versions
- Use attribute directives (ngClass, ngStyle) to modify element appearance and behavior
- Create custom attribute directives
- Create custom structural directives
- Understand HostBinding and HostListener for directive-host element interaction

## Key Concepts
Directives extend HTML with custom behavior. Structural directives alter the DOM layout by adding or removing elements, while attribute directives change the appearance or behavior of existing elements. Angular's newer control flow syntax (@if, @for, @switch) provides a template-native alternative to the traditional structural directive microsyntax. Custom directives encapsulate reusable DOM manipulation logic.

## Topics Covered
- Structural directives (*ngIf, *ngFor, *ngSwitch)
- New control flow syntax (@if, @for, @switch)
- Attribute directives (ngClass, ngStyle)
- Creating custom attribute directives
- Creating custom structural directives
- HostBinding and HostListener
