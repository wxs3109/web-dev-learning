# Capstone Projects

Mini-projects that integrate skills from across all pillars. Complete these after finishing the relevant learning modules to practice applying concepts in realistic scenarios.

## Projects

| # | Project | Skills Practiced |
|---|---|---|
| 01 | [Responsive Portfolio](01-responsive-portfolio/) | Semantic HTML, Flexbox/Grid, responsive design, CSS animations, accessibility |
| 02 | [Interactive Todo App](02-interactive-todo-app/) | DOM manipulation, events, localStorage, ES6 classes, error handling |
| 03 | [API Dashboard](03-api-dashboard/) | Fetch API, async/await, error handling, dynamic DOM, CSS Grid |
| 04 | [Angular Full App](04-angular-full-app/) | Components, routing, forms, HTTP client, RxJS, services, state management, testing |

## Approach
Each project folder contains a README describing the project requirements, features to implement, and which learning modules it draws from. Build these from scratch — no starter code is provided.
