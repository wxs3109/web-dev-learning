# Front-End Web Development Learning

A structured, progressive learning repository covering the core front-end technologies: **HTML/CSS**, **JavaScript/TypeScript**, and **Angular**.

## Repository Structure

| Section | Description |
|---|---|
| [01-html-css/](01-html-css/) | HTML & CSS fundamentals through advanced techniques |
| [02-javascript-typescript/](02-javascript-typescript/) | JavaScript fundamentals through advanced TypeScript |
| [03-angular/](03-angular/) | Angular framework from setup to production patterns |
| [04-web-fundamentals/](04-web-fundamentals/) | Cross-cutting web concepts (browser, performance, tooling) |
| [projects/](projects/) | Capstone mini-projects integrating skills from all pillars |

## Recommended Learning Order

```
01-html-css (modules 01-08)  ──>  02-js-ts (modules 01-10)
        │                                │
01-html-css (modules 09-12)       02-js-ts (modules 11-14, TypeScript)
        │                                │
        └────────────┬───────────────────┘
                     │
             04-web-fundamentals (anytime, reference material)
                     │
             03-angular (modules 01-14)
                     │
               projects/
```

1. **HTML & CSS basics** (structure and styling) — modules 01-08
2. **JavaScript fundamentals** (variables, types, functions, DOM, async) — modules 01-10
3. **Advanced CSS** (animations, variables, Sass, architecture) — modules 09-12
4. **TypeScript** (types, interfaces, generics, decorators) — modules 11-14
5. **Web fundamentals** (browser internals, performance, build tools) — anytime as reference
6. **Angular** (components, services, routing, forms, RxJS, testing) — modules 01-14
7. **Capstone projects** — integrate everything

## Prerequisites

- A code editor (VS Code recommended)
- A modern browser (Chrome/Edge with DevTools)
- Node.js and npm installed
- Git basics

## Folder Structure Convention

Each topic folder contains:
- `README.md` — Learning objectives, key concepts, topics covered, prerequisites, resources
- `examples/` — Reference code demonstrating concepts
- `exercises/` — Practice problems (starter files and solutions)
- `notes/` — Written notes (for conceptual topics in web-fundamentals)
